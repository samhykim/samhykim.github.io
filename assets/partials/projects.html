RFID Door Reader

Description:

An RFID reader was hooked up with an Arduino UNO in order to act as a door lock. Also attached was a powerful servo, which could only move when the correct unique RFID tag was present. However, this required the tag to be held at most 3cm away from the reader, which proved to not be very accessible.

Bluetooth Door Lock

Description:

To increase the distance from which the arduino could be accessed, a bluetooth module was attached. Using the Android app Ardudroid (credits: Hazim Bitar), I customized the arduino file provided to serve the application of unlocking a door. The mechanics remained the same as in the RFID door reader project.
